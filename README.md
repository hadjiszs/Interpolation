Interpolation
=============

Fast value interpolation by space decomposition into simplexes.
Space decomposition was done with an octree variant for simplexes

### Compilation

```shell
mkdir build
cd build
cmake ..
make
```

### Run

```
$ ./build/src/exec -f [PATH_FILE_CSV]
```

Example :

```
$ ./build/src/exec -f data/data11.csv
```
